## Setting up AWS Environment variables
Configure your environment variables with your IAM user credentials 
```
AWS_ACCESS_KEY=<YOUR_ACCESS_KEY_ID>
AWS_SECRET_ACCESS_KEY=<YOUR_SECRET_ACCESS_KEY_ID>
```
